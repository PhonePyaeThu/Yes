## Installation 

```javascript
git clone https://github.com/PhonePyaeThu/Yes/
cd Yes
pip install -r requirements.txt
python3 url.py
```


## Usage

Read the following statements 

- python3 url.py ( for only data add )
- python3 start.py ( to edit configs )
- python3 bot.py ( to start bot )
